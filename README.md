## About This Repository

This repository contains laboratory experiments for the **Containerization and DevOps** course.  
The experiments focus on understanding:

- Virtual Machines using **VirtualBox and Vagrant**
- Containers using **Docker and WSL**
- Differences in performance, resource utilization, and isolation
- Hands-on deployment of services like **Nginx**

Each experiment includes:
- Objective
- Theory
- Step-by-step commands
- Screenshots
- Results and conclusion
